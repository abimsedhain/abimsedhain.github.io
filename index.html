My Projects



  • COVID-19 Nepal

    A website made to display the total number of COVID-19 cases in Nepal. This shows the total confirmed cases, with recovered number, death number and total test conducted. The number is displayed through chart as well. In addition to that, this also shows the global COVID-19 cases.

  • Air Quality Index

    A twitter bot that tweets the Air Quality Index of Kathmandu Valley in 4 hours interval. This uses the airvisual api to get the data, NodeJS is used for backend. Twitter api and heroku is used to tweet the details.

  • Digital Libarary OCR Project

    A digital library for old nepali books. This project is an initiation to preserve old nepali story and poem books. Optical Character Recognition and OpenCV was used to scan and store in text files.

  • Ma Bolxu (I will talk)

    An android application built to help the differently abled tourist communicate in Nepali. This was build as part of a Hackathon with the theme of Visit Nepal 2020.

  • Recreation Center Management Web Application

    This application was created as final year project. This allowed a gym facility to record facility attendees by scanning the QR Codes of their cards. Android Library was used to handle the QR Code Generation and scanning, NodeJS was used for the backend.
